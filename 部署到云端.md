# 班级作业提交系统 - 免费部署到云端

> 全程无需信用卡，数据持久化不丢失，24小时在线

---

## 整体流程

```
你的代码 → GitHub → Hugging Face Spaces（托管应用） + Supabase（托管数据） → 全班都能访问
```

**费用：0 元。** 需要注册 2 个免费账号（GitHub + Hugging Face + Supabase），都不需要信用卡。

---

## 第一步：把代码上传到 GitHub

### 1.1 注册 GitHub
打开 https://github.com → 点 **Sign up** → 用邮箱注册

### 1.2 创建仓库
登录后点右上角 **+** → **New repository**
- Repository name: `homework-system`
- 点 **Create repository**

### 1.3 上传代码
在电脑上打开终端（CMD），逐条执行：

```bash
cd C:\Users\colorful\Desktop\test
git init
git add .
git commit -m "init"
git branch -M main
git remote add origin https://github.com/你的GitHub用户名/homework-system.git
git push -u origin main
```

> 把 `你的GitHub用户名` 换成你的实际用户名（如 `zhangsan`）
> 执行 `git push` 时会弹出浏览器让你登录 GitHub 授权

---

## 第二步：注册 Supabase（免费数据库）

### 2.1 注册
打开 https://supabase.com → 点 **Start your project** → 用 GitHub 登录

### 2.2 创建数据库项目
点 **New project**：
- **Name**: `homework-system`
- **Database Password**: 点 **Generate a password** → 把密码**复制保存好**
- **Region**: 选 **Singapore**（新加坡，离中国最近，速度快）
- **Pricing Plan**: 选 **Free**（免费）

点 **Create new project**，等待约 1 分钟创建完成。

### 2.3 获取数据库连接串
项目创建完成后：
1. 点左侧菜单 **Project Settings**（齿轮图标）
2. 点 **Database**
3. 找到 **Connection string** → 点 **URI** 旁边的 **Copy**
4. 得到一个以 `postgresql://` 开头的长字符串

> **这就是 DATABASE_URL，非常重要，保存好！**

---

## 第三步：部署到 Hugging Face Spaces（免费，24小时在线）

### 3.1 注册 Hugging Face
打开 https://huggingface.co → 点 **Sign Up** → 用邮箱注册（**不需要信用卡**）

### 3.2 创建 Space
1. 登录后点右上角头像 → **New Space**
2. 填写：
   - **Space Name**: `homework-system`（会自动生成网址）
   - **License**: 随便选（如 `mit`）
   - **Space SDK**: 选 **Docker**
   - **Docker Template**: 选 **Blank**
   - **Space Hardware**: 选 **Free**（免费的 CPU 基础套餐）
3. 点 **Create Space**

### 3.3 上传代码
在刚创建的 Space 页面：
1. 点 **Settings**（设置标签）
2. 往下翻到 **Repository**，会看到类似 `https://huggingface.co/spaces/你的用户名/homework-system`
3. **方式一（推荐）：同步 GitHub**
   - 点 Settings → **Link GitHub Repository**
   - 选你的 `homework-system` 仓库
   - 以后 GitHub 更新代码，Space 自动重新部署
4. **方式二：直接上传**
   - 点 **Files** → **Add file** → **Upload files**
   - 把整个项目文件夹拖进去上传

### 3.4 设置环境变量
在 Space 页面点 **Settings** → **Repository Secrets**：
1. 点 **New secret**
   - Name: `DATABASE_URL`
   - Value: 粘贴从 Supabase 复制的连接串
2. 再点 **New secret**
   - Name: `PORT`
   - Value: `7860`（Hugging Face 默认端口）

### 3.5 等待部署
点顶部的 **App** 标签，会看到构建日志。
第一次部署需要 3-5 分钟，构建完成后就能看到登录页面了。

部署成功后，你的系统地址是：
```
https://你的用户名-homework-system.hf.space
```

## 第四步：开始使用

2. 用 `admin` / `admin123` 登录
3. 把这个网址发到班级群，**全班同学在任何地方都能访问**

## 常见问题

### Q: 长时间没人访问会休眠吗？
A: 免费版超过 48 小时无人访问会休眠，醒来需要 30 秒。**但你们班天天用就不会休眠。**

### Q: 数据会丢失吗？
A: **不会。** 数据存在 Supabase 免费数据库里，应用重启或更新都不影响。

### Q: 如何更新代码？
如果设置了 GitHub 同步：
```bash
# 修改代码后执行
git add .
git commit -m "更新了什么"
git push
```
Hugging Face 会自动重新部署。

### Q: 如何查看服务器日志？
在 Space 页面点 **Settings** → **App** → 点右上角 **三** 图标 → **View logs**

### Q: 想绑定自己的域名？
Space 的 **Settings** → **Custom Domain** 可以绑定（非必需）
