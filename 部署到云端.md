# 班级作业提交系统 - 免费部署到云端

> 全程无需信用卡，数据持久化不丢失，24小时在线

---

## 整体流程

```
你的代码 → GitHub → Zeabur（托管应用） + Supabase（托管数据） → 全班都能访问
```

---

## 第一步：把代码上传到 GitHub

1. 打开 https://github.com 注册账号（邮箱即可）
2. 登录后点绿色的 **"New"** 按钮 → 创建仓库
3. 仓库名填 `homework-system` → 点 **Create repository**
4. 在电脑上打开终端（CMD），执行：

```bash
cd C:\Users\colorful\Desktop\test

git init
git add .
git commit -m "init"
git branch -M main
git remote add origin https://github.com/你的用户名/homework-system.git
git push -u origin main
```

> 把 `你的用户名` 换成你的 GitHub 用户名

---

## 第二步：注册 Supabase（免费数据库）

Supabase 提供免费的 PostgreSQL 数据库（500MB），**不需要信用卡**。

1. 打开 https://supabase.com
2. 点 **"Start your project"** → 用 GitHub 登录
3. 创建一个组织（随便取名）
4. 点 **"New project"**
   - Name: `homework-system`
   - Database Password: **点 "Generate a password"** 并复制保存好！
   - Region: 选 **Singapore**（新加坡，离中国近）
   - Pricing Plan: **Free**
5. 等一两分钟，项目创建完成后：
   - 点左侧 **Project Settings**（小齿轮图标）
   - 点左边 **Database**
   - 找到 **Connection string** 下的 **URI**（以 `postgresql://` 开头）
   - 点 **Copy** 复制

这个长链接就是 **DATABASE_URL**，保存好，下一步要用。

> 如果提示要绑卡，选 "Start with Free plan"，不需要信用卡

---

## 第三步：注册 Zeabur（部署应用）

1. 打开 https://zeabur.com
2. 点 **Sign in** → 用 GitHub 登录（和 GitHub 同一个账号就行）
3. 登录后点 **"创建项目"**
4. 取个名字（如 `homework`）
5. 点 **"导入你的代码"**
   - 选 GitHub
   - 如果提示安装 Zeabur 到 GitHub，点同意
   - 选刚才创建的 `homework-system` 仓库
   - 点 **导入**
6. Zeabur 会自动检测到 Dockerfile 开始部署
7. 等待部署完成（约 3 分钟）

---

## 第四步：设置环境变量

部署完成后，还需要告诉系统连接数据库：

1. 在 Zeabur 项目页面，点你的服务
2. 点顶部的 **"Environment Variables"**（环境变量）
3. 点 **"Add"**
   - Key: `DATABASE_URL`
   - Value: 粘贴刚才从 Supabase 复制的连接串
4. 点 **"Deploy"**（重新部署）

等重新部署完成后，你的系统就上线了！

---

## 第五步：获取网址并分享

1. 在 Zeabur 项目页面，点你的服务
2. 顶部会显示一个网址，如 `https://homework-system.zeabur.app`
3. 点开它，应该能看到登录页面
4. 用 `admin` / `admin123` 登录
5. 把这个网址发到班级群

> 可以点 Zeabur 里的 **"Custom Domain"** 绑定自己的域名（非必须）

---

## 日常管理

### 查看数据
登录后台 → 作业管理（查看提交）/ 学生管理（查看注册学生）

### 下载数据
在作业详情页可以导出 CSV 和下载所有文件压缩包

### 修改管理员
登录后用 `admin` 账号可以在"系统管理"里添加/删除管理员

### 更新代码
```bash
git add .
git commit -m "修改了什么"
git push
```
Zeabur 会自动重新部署

---

## 故障排查

**首页能打开但登录提示"无效的登录凭证"**
→ 检查 DATABASE_URL 环境变量是否设置正确

**页面白屏/502错误**
→ 在 Zeabur 点 "Logs" 查看日志

**想重设管理员密码**
→ 设置环境变量 `RESET_ADMIN=true` 并重新部署（会自动创建默认 admin 账号）

**数据一开始是空的**
→ 首次部署后，用 admin / admin123 登录即可。系统会自动创建默认管理员。
