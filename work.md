# 班级作业文件夹提交系统 - 项目文档

## 一、项目背景与目标
为解决班级作业提交过程中，文件分散、版本混乱、收集统计困难等问题，开发一套**轻量、易用、本地化部署**的作业提交系统。
- 管理员可创建作业、发布提交链接、一键导出提交名单与文件
- 学生可通过链接提交作业文件夹，无需注册账号
- 所有提交自动归档，方便管理员后续整理与评分

---

## 二、核心功能需求

### 1. 管理员功能
1.  **作业创建**
    - 可设置作业名称、截止日期、作业说明/要求
    - 生成唯一的作业提交链接（本地访问即可）
    - 支持编辑、删除已创建的作业
2.  **提交管理**
    - 实时查看每个作业的提交状态（已提交/未提交名单）
    - 支持导出提交记录为 CSV 文件（含姓名、学号、提交时间）
    - 支持一键打包下载所有提交的作业文件夹
3.  **系统设置**
    - 可配置提交文件总大小上限（单作业最大支持 500MB）
    - 可自定义文件存储根目录路径

### 2. 学生功能
1.  **作业提交**
    - 通过管理员发布的链接进入提交页面
    - 填写姓名、学号等必要信息
    - 选择本地作业文件夹上传，支持拖拽上传
    - 实时显示上传进度与提交结果
2.  **状态查看**
    - 提交后可查看提交状态（成功/失败/待审核）
    - 同一作业支持重复提交（以最后一次为准）

## 三、技术栈与架构设计

### 1. 技术栈
| 模块   | 技术选型          | 说明                          |
|--------|-------------------|-------------------------------|
| 前端   | Vue 3 + Vite + Element Plus | 实现页面可视化、表单、上传进度条 |
| 后端   | Python + FastAPI  | 处理文件上传、数据存储、接口逻辑 |
| 存储   | 本地文件系统 + JSON | 作业配置、提交记录、文件存储    |
| 打包   | PyInstaller       | 将后端打包为可执行文件，方便部署 |
